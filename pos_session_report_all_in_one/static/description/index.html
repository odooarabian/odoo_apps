POS Session Report

Print your Session Summary using our POS Session Report Module. This module allows you to print or send POS Session Summary via email. This session summary contains details like basic session details, statement summary, statement details, order summary, product sale details etc.

Features
  • Seller can print Point Of Sale Session Summary from backend as well as from running POS Session.
  • User can send Point Of Sale Session Summary via emai.
  • Seller can print Session Summary in both Xml Recipt (POSBOX) or pdf format in POS Session depending upon backend POS Configuration.
  • No complex configuration required.
  • Session Summary contains following details:
    • Session Basic Details like Responsible, Point of Sale , Opening Date, Closing Date, Total Sale etc.
    • Complete Statement Summary.
    • Statement Details.
    • POS Order Details.
    • Product Summary.
  • POS Session View
  • Change Log
